## Cheat Sheet — Git Commands You’ll Use Every Day

|What you want to do     |Command                       |
|------------------------|------------------------------|
|Start a new day’s branch|`git checkout -b day-XX`      |
|See what’s changed      |`git status`                  |
|Stage all changes       |`git add .`                   |
|Stage one file          |`git add week1/day1.py`       |
|Commit with a message   |`git commit -m "your message"`|
|Push to GitHub          |`git push origin day-XX`      |
|Switch back to main     |`git checkout main`           |
|Pull latest from GitHub |`git pull origin main`        |
|See all your branches   |`git branch`                  |
|See your commit history |`git log --oneline`           |
